Organization is of the essence.

This is the first group project of the 42 CommonCore. The idea is to recreate a shell with a specific set of built-in functions. To succed in this project there are three essential tasks that need to be completed:

1. The shell must be able to recieve input from the user and parse it to guarantee that there are no syntax errors.

2. The program must, then, organize the commands that the user input and define the order in which those commands are executed.

3. The shell must run the commands in order and display or output the result.
